REST Server Auth API

Available operations

api/auth/register

Register the user into the system.

Method: POST

Response codes:

  • 200 OK (Successful operation)
  • 400 Bad request (Errors validating the input)
  • 500 Internal server error (Internal errors)

Request example:


        {
          "name": "Username",
          "email":"username@email.com",
          "password":"password"
        }

Response example (successful operation):


        {
          "user": {
              "id": "663c8479d14bdd3a148be4ee",
              "name": "Username",
              "email": "username@email.com",
              "role": [
                  "USER_ROLE"
              ]
          },
          "token": "YOUR_JWT_TOKEN"
        }

Response example (unsuccessful operation):


        {
          "error": "Email already exists"
        }
